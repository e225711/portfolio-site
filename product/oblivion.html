プロジェクトの概要

OBLIVIONは、翻訳時に単語カードを効率的に作成し、語彙を記憶できるスマホアプリです。このアプリは、翻訳と単語暗記をシームレスに統合し、学習者が効率的に語彙を増やすことを目指しています。

開発の背景と目的

OBLIVIONは、以下の課題を解決するために開発されました。

・翻訳してもすぐに忘れてしまう

・単語帳を作るのが面倒である

使用した技術スタック

TypeScript、React Native、Expo、DeepL API

技術選定の理由

React Nativeを選定した理由は、クロスプラットフォーム対応が可能であり、iOSとAndroidの両方で動作するアプリを効率的に開発できるためです。 また、TypeScriptを使用することで、コードの信頼性とメンテナンス性が向上しました。

工夫、苦労した点

コードが増えるに連れてファイルやフォルダがごちゃごちゃしてしまい、Atomic Designを取り入れて設計を見直しました。また、共同開発者とのコミュニケーションにも力を入れ、スムーズに開発を進めました。さらに、App Storeへのアップロードの際の審査対応にも苦労しましたが、現在申請中です。

学んだことや挑戦

知識がなくても調べながら作成することでアプリを完成させることができ、自信を持つことができました。また、共同開発者との連携やコミュニケーションの重要性を学び、将来的にエンジニアとして活躍する際にも役立つスキルを得ることができました。
